# Информационная система химчистки (Вариант 25)

Курсовой проект по дисциплине «Базы данных». Приложение представляет собой клиент-серверную систему управления химчисткой, реализованную на архитектурном шаблоне **MVC** с использованием **C++ / Qt 6** и СУБД **PostgreSQL**.

---

## Требования для запуска

- **СУБД**: PostgreSQL версии 15 или 16 (должен быть запущен локально на порту `5432`).
- **Клиент**: Windows 10 / 11 (для запуска готовой сборки) или macOS/Linux (для сборки из исходников).

---

## Инструкция по развертыванию и запуску

### Шаг 1. Настройка базы данных в PostgreSQL

Приложение жестко настроено на подключение к локальной базе данных `dryclean` под пользователем `user` без пароля.

1. Откройте консоль PostgreSQL (`psql`) или любой GUI-клиент (pgAdmin, DBeaver) под учетной записью суперпользователя (`postgres`).
2. Создайте пользователя `user`:

   ```sql
   CREATE USER "user" WITH SUPERUSER PASSWORD '';
   ```

   _(Пароль оставляется пустым для беспарольного доверенного входа на локальной машине)_.
3. Создайте база данных `dryclean`, владельцем которой является созданный пользователь:

   ```sql
   CREATE DATABASE dryclean OWNER "user";
   ```

4. Импортируйте структуру таблиц, триггеры и представления из файла схемы. В консоли выполните:

   ```bash
   psql -U user -d dryclean -f app/schema.sql
   ```

   _(Либо откройте содержимое файла `app/schema.sql` в pgAdmin/DBeaver и выполните весь скрипт)._

---

### Шаг 2. Запуск клиентского приложения

#### Вариант А: Запуск готового релиза (Рекомендуется для Windows)

1. Скачайте архив сборки из вкладки **Releases** (артефакт `dryclean-windows-x64`).
2. Распакуйте архив в любую удобную папку.
3. Запустите файл `dryclean.exe`. Приложение автоматически подключится к вашей локальной базе данных.

#### Вариант Б: Сборка из исходного кода (Для разработчиков)

1. Установите **Qt SDK 6.x** и среду **Qt Creator**.
2. Убедитесь, что в системе установлен драйвер `QPSQL` (поставляется вместе с Qt).
3. Откройте в Qt Creator файл проекта: `app/dryclean.pro`.
4. Настройте комплект сборки (Kit) под вашу систему (например, Desktop Qt 6.x.x MinGW/MSVC/Clang).
5. Запустите сборку и выполнение проекта (клавиши `Ctrl+R` или кнопка **Run**).
